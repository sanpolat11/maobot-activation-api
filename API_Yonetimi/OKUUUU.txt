   • Yeşil pencere AÇIK KALIR

│ 2. API_BASLAT.bat (YEŞIL) - Arka Plan                  │
└─────────────────────────────────────────────────────────┘

   • Pencereyi kapatabilirsin
   • Bazen çalışmayabilir

│ 3. API_KONTROL.bat (MAVİ) 🔍                            │
└─────────────────────────────────────────────────────────┘
   • ÇİFT TIKLA → API durumunu kontrol eder
   • Mavi pencere açılır
   • "API CALISIYOR!" veya "API CALISMIYOR!" gösterir

┌─────────────────────────────────────────────────────────┐
│ 4. API_DURDUR.bat (KIRMIZI) 🛑                          │
└─────────────────────────────────────────────────────────┘
   • ÇİFT TIKLA → API durur
   • Kırmızı pencere açılır
   • "API DURDURULDU!" mesajını gör

⚡ HIZLI BAŞLANGIÇ:
